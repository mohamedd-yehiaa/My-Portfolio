# Mohamed Yehia | Software Engineer Portfolio

A premium, highly responsive, single-page developer portfolio built from the ground up with **HTML5, CSS3, and Vanilla JavaScript**. Designed to showcase my expertise in Mobile Development (Flutter/Dart) and Cloud DevOps infrastructure without relying on heavy frameworks.



## ✨ Features

- **Zero-Dependency Architecture:** Built strictly with vanilla web technologies for maximum performance and maintainability.
- **Advanced CSS & BEM Methodology:** Structured CSS utilizing custom properties (CSS variables), modern grid layouts, and strict BEM naming conventions.
- **Dynamic Theme Toggling:** Seamless transition between Light and Dark modes using CSS variables, with `localStorage` persistence.
- **Performance Optimized:** Lazy-loaded images, deferred JavaScript execution, and lightweight SVG icons ensure lightning-fast load times.
- **Intersection Observer Animations:** Silky smooth, scroll-triggered reveal animations that bring the page to life without slowing down the browser.
- **Dynamic DOM Counters:** JavaScript-driven stat counters that automatically parse the DOM to tally up listed projects and skills.
- **Accessibility & SEO:** Semantic HTML, ARIA labels, focus states, and Open Graph meta tags built-in.

## 🛠️ Tech Stack

- **Structure:** Semantic HTML5
- **Styling:** Vanilla CSS3 (Flexbox, CSS Grid, Custom Properties, Media Queries)
- **Logic:** Vanilla JavaScript (ES6+)

## 🚀 Getting Started

Since this project uses pure static files, no build step or package manager is required. 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mohamedd-yehiaa/portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd portfolio
   ```
3. **Run locally:**
   Simply open `index.html` in your favorite web browser, or use a local development server like VS Code's Live Server extension for hot reloading.
